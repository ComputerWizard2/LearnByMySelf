1. 本程序基于登录对数据库的数据进行遍历并且进行获取输出在页面
2. 对于过滤器的实现功能主要在于对于编码的设置，以及对于监听器的对于用户的重复登录的监听的实现
3. 对于数据库的连接使用的是c3p0的建立，对数据库进行连接
4. 对于页面的数据的获取主要使用的是el的表达式以及jstl的表达式的使用
5. 总结：本项目的主要是对于正在学习j2e的同学一个较为综合的案例，希望大家通过这个案例更加理解过滤器以及监听器
c3p0 el jstl表达式的理解 

	第一步： 导入包这些包包括数据库的驱动，c3p0的jar包，以及el表达式的包
	第二步： 主要将所需要的包来建立包括链接数据库的util的工具包以及访问数据的dao层接口，过滤器以及监听器的package ,还有就是处理来自客户端的请求的servlet
	第三步：设置bean对象，我们这里不涉及复杂的对象目前就仅仅就是一个user对象
	第四步： 设计登录页面login.js
	第五步： 对过滤器以及监听器进行设置，过滤器主要对编码进行设置，而监听器对于用户的登录进行检测
			另外 我们可以在监听器对当前的在线人数进行检测。
